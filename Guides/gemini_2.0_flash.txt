Documentation
Get started
You can use Gemini 2.0 Flash in Vertex AI Studio's chat-like prompt editor. You can also use the command line or integrate the model in your application.

Before you begin
Enable the Vertex AI API.

For more information on getting set up on Google Cloud, see Get set up on Google Cloud.

Try Gemini 2.0 Flash in Vertex AI Studio (console)
To use Gemini 2.0 Flash in Vertex AI Studio, click Open Vertex AI Studio. In Vertex AI Studio, you can write a prompt then click Submit to view the output generated by Gemini 2.0 Flash.

Try Gemini 2.0 Flash (curl)
These instructions don't apply if you're using express mode for Google Cloud. If you're using express mode, follow the instructions for express mode instead.

To use Gemini 2.0 Flash with the command line interface (CLI), do the following:

Open Cloud Shell or a local terminal window with the gcloud CLI installed.

Configure environment variables by entering the following. Replace YOUR_PROJECT_ID with the ID of your Google Cloud project.

MODEL_ID="gemini-2.0-flash-001"
PROJECT_ID="YOUR_PROJECT_ID"
Send a prompt request by entering the following curl command:
curl \
  -X POST \
  -H "Authorization: Bearer $(gcloud auth application-default print-access-token)" \
  -H "Content-Type: application/json" \
  "https://us-central1-aiplatform.googleapis.com/v1/projects/${PROJECT_ID}/locations/us-central1/publishers/google/models/${MODEL_ID}:streamGenerateContent" -d \
  $'{
    "contents": {
      "role": "user",
      "parts": [
        {

For more information, see the Gemini API reference.

Try Gemini 2.0 Flash (Python)
These instructions don't apply if you're using express mode for Google Cloud. If you're using express mode, follow the instructions for express mode instead.

Before trying this sample, follow the Python setup instructions in the Google Gen AI SDK quickstart using client libraries.

To authenticate to Vertex AI, set up Application Default Credentials. For more information, see Set up authentication for a local development environment.

Install or update the Google Gen AI SDK. You can also install it in a virtual environment.
!pip3 install --upgrade --user google-genai
Send a prompt request. Replace YOUR_PROJECT_ID with your Google Cloud project ID.
from google import genai
from google.genai import types
client = genai.Client(
  vertexai=True, project="YOUR_PROJECT_ID", location="YOUR_LOCATION",
)
# If your image is stored in Google Cloud Storage, you can use the from_uri class method to create a Part object.
IMAGE_URI = "gs://generativeai-downloads/images/scones.jpg"
model = "gemini-2.0-flash-001"
response = client.models.generate_content(
  model=model,
